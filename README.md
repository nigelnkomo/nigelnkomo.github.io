# Jekyll Setup

This site is built with Jekyll.

## Setup

```bash
# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve

# Build for production
bundle exec jekyll build
```

## Directory Structure

- `_layouts/` - Page templates (default, page, paper, gallery)
- `_includes/` - Reusable components (header, footer, search)
- `_config.yml` - Jekyll configuration
- `assets/` - CSS, JS, images, fonts
- `index.md` - Homepage
- `about/`, `portfolio/`, `academic-writing/`, `photography/`, `search/` - Pages
